O projeto EasyShoes é um exercico dos princípios do framework
angular, desenvolvido com o objetivo de ser usado como portfolio
para atestar meus conhecimentos na ferramenta. Esse projeto exigiu
conhecimentos basicos dos conceitos primordiais do angular como a
criação e manipulação de componentes, services, uso de diretivas e
routing, além da estilização da interface utilizando a biblioteca bootstrap.
